Sobre Mim

 

Sou um desenvolvedor de software formado em Análise e Desenvolvimento de Sistemas pela Universidade Anhembi Morumbi.
Desde cedo, desenvolvi habilidades em projetos com Java e automatização com Python. Tenho experiência em explorar dados e gerar relatórios detalhados por meio do MySQL.
Desenvolvi vários projetos, entre os quais se incluem: um encurtador de URLs, algumas páginas estáticas com HTML e CSS, um gerador de senhas com Python integrado ao banco de dados para armazenar todas as senhas utilizadas pelos usuários. Além disso, meu portfólio inclui a criação de um site de avaliações de jogos com todas as operações CRUD, onde utilizei Node.js e Tailwind.
Para finalizar, elaborei um sistema desktop de cadastro de produtos totalmente desenvolvido em Java puro, utilizando a IDE NetBeans. Atualmente, sou estagiário na PagSeguro, mas já liderei uma equipe e juntos conseguimos fazer história na ótica Gassi. Treinei pessoas que se tornaram líderes de setor para outras unidades da ótica. Atualmente, estou focado em aprimorar minhas habilidades em desenvolvimento e automação com Python.
