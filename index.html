Sobre Mim

Olá, meu nome é Thiago. Sou desenvolvedor de software formado em análise e desenvolvimento de sistemas pela universidade Anhembi Morumbi.

Desde cedo, desenvolvi habilidades em projetos com Java e me destaquei na utilização versátil do Python. Tenho experiência em explorar dados e gerar relatórios detalhados através do Mysql.

Desenvolvi vários projetos, entre eles estão: Encurtador de url, algumas páginas estáticas com HTML e css, gerador de senhas com Python acoplado ao banco de dados armazenando todas as senhas utilizadas pelos usuários. Além disso, tenho em meu portfólio a produção de um site avaliações de jogos com todas as operações CRUD, onde utilizei node.js, Tailwind. Para finalizar elaborei um sistema de desktop de cadastro de produtos totalmente feito com java puro utilizando a IDE netbeans.

Estou atualmente como estagiário na PagSeguro, mas já liderei uma equipe e juntos conseguimos fazer história nas óticas Gassi, treinei pessoas que se tornaram líderes de setor para outras unidades da ótica.

Estou trabalho atualmente em minhas habilidades back-end e desenvolvimento de automação com python, estou aberto as novas oportunidades com desenvolvedor.

Se você está procurando um profissional apaixonado por desafios, inovação e compartilhamento de conhecimento, estou pronto para trazer resultados excepcionais à sua equipe. Vamos juntos alcançar novos patamares!
